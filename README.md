# ABI Studio / Online Schools — Feedback & Bug Reports

Этот репозиторий предназначен для публичных сообщений об ошибках, предложений и отзывов по сайту:

https://online-schools.net/

## О проекте

ABI Studio — онлайн-школа, где ученики проходят курсы онлайн-обучения через практические задания, AI-разбор и тренажёры навыков.

Формат подходит для дистанционного обучения с нуля: от программирования и веб-дизайна до работы с ИИ-ассистентами.

## Для чего этот репозиторий

Здесь можно:

* сообщить об ошибке на сайте;
* описать проблему с прохождением курса;
* предложить улучшение интерфейса;
* оставить идею по новым функциям;
* сообщить о непонятном задании, тексте или результате AI-разбора.

## Важно

Этот репозиторий не содержит исходный код проекта.

Исходный код ABI Studio / Online Schools находится в приватном репозитории и не публикуется.

Пожалуйста, не указывайте в обращениях:

* пароли;
* токены;
* данные банковских карт;
* персональные документы;
* приватные данные аккаунта;
* любую конфиденциальную информацию.

## Как сообщить об ошибке

Откройте вкладку **Issues** и создайте новое сообщение.

Желательно указать:

* ссылку на страницу, где возникла проблема;
* что именно произошло;
* какие действия привели к ошибке;
* что должно было произойти;
* устройство, браузер и операционную систему;
* скриншот или видео, если это возможно.

Спасибо за помощь в улучшении проекта.
